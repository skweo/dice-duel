# 骰誓 Dice Oath

一个面向中文玩家的暗黑中世纪骰子对战网页游戏。

玩家每回合掷 3 个骰子，把点数分配到攻击、防御、恢复三个行动槽中。敌人会提前显示本回合意图，玩家需要根据点数、敌人技能、当前血量和路线选择做取舍。

## 试玩

线上地址：

https://skweo.github.io/dice-duel/

本地也可以直接打开 `index.html` 游玩。

## 当前玩法

- 6 轮战斗，第 1 轮固定地点，第 2 到第 5 轮每轮三选一路线。
- 路线分为债路、血路、梦路，不同路线会遇到不同地点和敌人。
- 第 6 轮为 Boss 战，Boss 会根据玩家此前路线倾向决定。
- 击败 Boss 即胜利，并解锁对应结局。
- 敌人不仅数值递增，也会通过特殊技能改变骰子分配策略。
- 击败敌人后不会自动回血，当前血量会保留并进入下一场战斗。
- 剧情通过地点、敌人传闻、登场文本、败北文本和战斗记录碎片化展开。
