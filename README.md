# 骰誓 Dice Oath

一个面向中文玩家的暗黑中世纪骰子对战网页游戏原型。

每回合掷 3 个骰子，把骰子分配到攻击、防御、恢复三个行动槽中。敌人会提前显示本回合意图，玩家需要根据点数和敌人技能做取舍，尽可能获得更高连胜。

## 试玩

线上地址：

https://skweo.github.io/dice-duel/

本地也可以直接打开 `index.html` 游玩。

## 当前玩法

- 攻击：造成等同骰子点数的伤害。
- 防御：抵消敌人的本回合伤害。
- 恢复：每 2 点恢复 1 点生命。
- 已分配的骰子可以点击撤回，重新选择行动槽。
- 玩家拥有特殊技能“三相誓约”。
- 每个敌人都有不同特殊技能。
- 击败敌人后不会自动回血，当前血量会保留并进入下一场战斗。
