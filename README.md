# 骰誓 Dice Oath

一个面向中文玩家的暗黑中世纪骰子对战网页游戏。

玩家每回合掷 3 个骨骰，把点数分配到攻击、防御、恢复三个行动槽中。敌人也会同步掷出可见黑骰，玩家需要根据双方点数、敌人技能、当前血量、遗物和路线选择做取舍。

## 试玩

线上地址：

https://skweo.github.io/dice-duel/

本地也可以直接打开 `index.html` 游玩。

## 当前玩法

- 6 轮战斗，第 1 轮固定地点，第 2 轮自由立誓，第 4 轮才会出现一次裂隙改道机会。
- 路线分为债路、血路、梦路，不同路线会遇到不同地点和敌人；如果中途改道，会损失生命并留下违誓痕迹。
- 第 3 轮和第 5 轮会沿当前路线自动深入，不会每回合都给玩家无代价换路。
- 第 6 轮为 Boss 战，Boss 会根据玩家此前路线倾向决定。
- 击败 Boss 即胜利，并解锁对应结局。
- 开局可选择 2 名角色，击败不同 Boss 后会解锁特定新角色。
- 每名角色拥有独立生命上限、技能、性格与背景故事。
- 敌人每回合会掷出可见黑骰，而不再只显示攻击意向；敌人技能会在黑骰基础上改变最终伤害。
- 攻击、防御、恢复槽都有独立特性：双骰攻击破甲、防御完全格挡会反击、双骰恢复会强化祷疗。
- 击败普通敌人会从该敌人的道具池中随机获得 1 件遗物；金色永久遗物会后续自动生效，红色一次性遗物可点击使用。
- 获得遗物时会出现醒目的“新获得”提示，避免奖励被战斗日志淹没。
- 图鉴分为敌人与遗物，遭遇敌人或获得遗物后会逐步解锁档案。
- 敌人不仅数值递增，也会通过特殊技能改变骰子分配策略。
- 击败敌人后不会自动回血，当前血量会保留并进入下一场战斗。
- 剧情通过地点、敌人传闻、登场文本、败北文本和战斗记录碎片化展开。
- 初始选人界面改为更明亮的收藏卡高级感 UI，玩家与敌人画像全部重做为差异化剪影/徽章卡牌。
- 掷骰区新增伪 3D 投骰剧场：主角伸手抛出 3 枚骰子，骰子会飞出、落桌、反弹并停在台面上。
