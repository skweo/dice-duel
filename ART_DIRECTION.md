# Dice Oath Art Direction

This document keeps the game's visual direction consistent before and after replacing SVG portraits with generated raster artwork.

## Core Style

Dark medieval adventure, cursed chapel gambling table, candlelight, weathered iron, bone dice, old parchment, rust, blood-red wax seals, worn leather, and occult heraldry.

The tone can borrow broad influences from tactical dark fantasy, gothic religious horror, mythic folklore, and strange otherworldly character design. Do not copy specific characters, costumes, symbols, logos, or exact visual styles from existing franchises.

## UI Prompt

Use case: ui-mockup
Asset type: browser game interface reference
Primary request: a dark medieval dice duel game UI with two character portrait cards, HP bars, three large dice, and three action slots for strike, ward, and mend
Style/medium: polished 2D fantasy game UI concept art
Scene/backdrop: cursed chapel gambling table, candlelight, old stone, parchment, iron trim, blood-red seals
Composition/framing: portrait-friendly web game layout, readable on mobile and desktop
Color palette: blackened brown, burnt umber, bone ivory, old gold, blood red, dull steel
Constraints: no logos, no readable franchise symbols, no copyrighted characters, no watermark
